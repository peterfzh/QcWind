PC版本网页前端开发小助手

# 使用说明
  1.下载QcWind到您的Web项目中，然后再使用的页面内加上如下的引用
  
  <script type="text/javascript" src="/您的引入的路径/QcWind.js"></script>

  -----

  2.类似Jquery的加载完成事件

  <script type="text/javascript">    


    QcWind.ready(function(){

		//您需要做的操作,类似于Jquery.ready

	});

   </script>
